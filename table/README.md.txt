# 編輯材質
請進入 Shading 模式

# 節點介紹
## 基本Material節點
- [Shader] Principled BSDF 節點 : 可以視為主要的Material輸入集合入口節點 <br>
- [Output] Material Output 節點 : 最終將Material輸出的節點

## image圖片節點
- [Texture] Image Texture 節點 : 需要上傳圖片的，用這個就對了 <br>
